Because of JavaScript security restrictions enforced by some browsers, this page needs to be served by a Mingle instance or a webserver so that the host and port are the same as the Mingle instance.

You can do this easily in a development environment by moving this git repo to the MINGLE_DIR/public folder. It can then be accessed here.
